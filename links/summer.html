Summer Experience - Konstantin

Summer Intern - Concertio


Concertio is a tech startup based in New York that specializes in optimization. This summer, I worked at Concertio as a Software Engineer intern.
My task is to build an extension for the Microsoft Visual Studio Code that supports the main product of the company - Concertio Optimizer Studio.
To do that, I had to learn a lot of new things: programming language TypeScript, VS Code Extension API, Optimizer Studio CLI, Optimizer Studio API, Jira, Bitbucket, and many more!
I was involved in the design process, engineering, testing, and improving the extension for the best user experience. It was my first tech internship, and I'm so proud of the work I've done.
This experience made me a better developer and a better professional.
